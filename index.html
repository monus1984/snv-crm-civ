🚀 Déploiement GitHub → Netlify

Poussez ce fichier HTML vers un repo GitHub — Netlify le redéploie automatiquement à chaque commit.

🟢
Configuration GitHub
Configuration Netlify (optionnel)
🌐 Site :  ·  dernier commit ↗
📖 Guide de mise en place (première fois)
1. Créer le repo GitHub — github.com › New repository (public ou private, nom libre)
2. Créer un token GitHub — github.com › Settings › Developer settings › Personal access tokens › Fine-grained ou Classic avec scope repo
3. Premier push — renseignez les champs ci-dessus et cliquez "Pousser sur GitHub". Le fichier apparaît dans le repo.
4. Connecter Netlify — netlify.com › Add new site › Import from Git › choisir le repo. Publish directory : / (racine). Build command : vide.
5. Deploy Hook (optionnel) — Netlify › Site settings › Build & deploy › Build hooks › "Add build hook". Collez l'URL ici pour pouvoir forcer un redéploiement sans commit.
À chaque clic sur "Pousser sur GitHub", Netlify détecte le commit et redéploie automatiquement en ~30 secondes.
